# Healthcare Patient Portal

A responsive Healthcare Patient Portal built using HTML, CSS, and JavaScript.

## Features

- Dashboard
- Book Appointments
- Search Appointments
- View Medical Records
- View Lab Results
- View Prescriptions
- Edit Patient Profile
- Local Storage for Appointments and Profile
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
